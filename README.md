# Henlo Hooman!

HenloHooman is an interactive telegram doggo bot, which aims to spread PAWS-itivity to the world!

Users are able to request for cute and funny

1. stickers
2. gifs
3. pictures
4. jokes

from the bot!

Share the laughter and joy with **@henlohooman_bot** on Telegram!
